# Privacy And GDPR/DPIA Working Drafts

This folder contains working drafts for privacy, retention, and production-readiness review.

These documents are not a legal conclusion and must not be treated as proof that the application is GDPR-compliant. Before use with real patient data, the hospital DPO, legal service, information security team, and clinical IT owner must review and approve the application, Firebase configuration, access rules, retention settings, operating procedures, and incident response process.

- [Retention policy](retention-policy.md)
